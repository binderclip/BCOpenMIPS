### 宏定义 [defines.v](./defines.v)

里面是一些整个功能会用到的宏定义，主要是一些 1、0 值的定义，固定的地址、长度。

### PC 模块

取指阶段取出指令存储器的指令，同时为了取下一条指令 PC 值自增。包括 PC、IF/ID 两个模块。

#### [pc_reg.v](./pc_reg.v)

简单的 PC 指针随时钟自加。

#### [if_id.v](./if_id.v)

简单的寄存器值传递。

## 小知识点

### 赋值

`pc <= 32'h00000000;` 为非阻塞赋值。

而使用 `==` 的时候为阻塞赋值。前面还可以加延时的时间。

```
#5 rst = 1'b0;
```
